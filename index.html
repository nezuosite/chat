📖 ガイド ＆ セキュリティ仕様

🔑 1. 概要と使用方法 ・表示名と任意のトリップキー、部屋ID、共通暗号化パスワードを入力して「入室 / 作成」を押します。
・入室後に上部の「🔗 URL」を押して招待リンクをコピーし、対話相手に安全な手段で共有してください。
・不特定の第三者の侵入を防止するため、入室後にホストはルーム上部で「オープン / 承認制（LOCKED）」を自由に切り替えることができます。
🔒 2. エンドツーエンド二重暗号化（E2EE） ・本システムは、ブラウザネイティブの暗号化APIである Web Cryptography API (AES-GCM 256bit) を採用しています。
・発言内容・ニックネーム・画像データは全て送信前にローカルで暗号化されるため、ネットワーク経路やデータベース管理サーバ上にはデタラメな暗号化バイナリしか存在しません。
・解号キーとなる共通パスワードはサーバに送信されないため、運営者を含め、対話者以外の誰も解読・復元することは数学的に不可能です。
🚪 3. 自動物理消去（Wipe）設計 ・本システムは、最後の一人が部屋を退室する、または設定した寿命の制限時間に達した瞬間に、サーバからその部屋に関わるチャットデータ・登録情報が物理的に完全削除されます。
・サーバ内にバックアップや残余ログなどは一切保存されないため、一度消去されたログを後から復元することは、いかなる技術を用いても完全に不可能です。
